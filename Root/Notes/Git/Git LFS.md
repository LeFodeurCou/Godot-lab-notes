>Git LArge File Storage

Used for large files like assets

```bash
git lfs install # Not a git command ???
git lfs track "*.png" "*.jpg" "*.ogg" "*.wav" "*.mp4"  
git add .gitattributes  
git commit -m "Enable Git LFS for assets"  
git push
```
